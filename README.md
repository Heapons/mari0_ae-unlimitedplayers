# Features
## Uncapped Player Slots
You can play up to... however many players you want to play with! The game is no longer limited to 4 players only.

The only limit here is your <s>imagination</s> ***hardware***.

> [!TIP]
> - Press `1` to quickly set to player count to one.
> - Press `2` to double the current player count.

## Cheats
### Friendly-Fire

When enabled, players are able to deal damage to each others with their power-ups as if they were enemies.

> [!NOTE]
> **Player Collisions** is required to allow damage through stomping and starmen.

## Netplay
- New setting: Friendly-Fire.
- The Server IP and Port are now printed once in the chat upon creating a game.
- The clipboard now gets fetched from your OS' clipboard, meaning that you can paste the Server Address more easily.
